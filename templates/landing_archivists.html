03.

Frequently Asked Questions

What archiving methods does Speculum support?
Speculum supports three archiving methods: wget for complete website mirroring with recursive crawling, yt-dlp for YouTube channels and video content, and SingleFile for JavaScript-heavy single-page applications.
Can I access the archive programmatically?
Yes, Speculum provides a REST API for searching the archive, checking crawl status, and accessing metadata. The API supports JSON responses and is CORS-enabled for web applications.
How does Speculum handle large websites?
Speculum uses rate-limited recursive crawling with configurable depth limits. Large sites are processed in the background with progress tracking and automatic retry on failures.
Is there Wayback Machine integration?
Yes, Speculum integrates with the Internet Archive's Wayback Machine. You can submit URLs for preservation to both Speculum's local archive and the Wayback Machine simultaneously.
